## Purpose
This website was created for Assignment 2.  
It includes:
- A personal homepage (education, skills, experience, contact)
- A visualization portfolio page using SVG generated with JavaScript

## Technologies Used

### HTML
Used to structure the website content including:
- Sections
- Headings
- Lists


### CSS
Used for styling and layout.

Styling choices include:
- Very light orange background for a soft professional look
- Large black main header (h1)
- Smaller clean body text

### JavaScript
Used for:
- Generating SVG visualizations (bar chart + skyline)
- Basic form validation (contact form interactivity)

### SVG
Used to create:
- A bar chart showing hours watched across 7 days
- A stylized city skyline with buildings, windows, sky, and moon

## Website Structure

A2-Personal-Website/
- index.html (homepage)
- visualizations.html (SVG visualization portfolio)
- style.css (styling)
- main.js (basic interactivity + form validation)
- vis.js (SVG drawing and data visualization)
- README.md (project explanation)
